Summary:
The provided documentation for the Financial Consolidation product includes detailed instructions on user settings, workflow management, currency exchange rates, period locking, and company management. The User Settings section covers profile customization, period selection, and language settings. Workflow Management explains how to manage tasks, view Gantt and Pie charts, and export data. The Currency Exchange Rates section details how to enter, import, and export exchange rates for different consolidation periods. The Period Locking section outlines how to set dates to disable the import or input of specific data types. The Company Management section provides steps to add and manage companies within the group structure, including additional company data and ownership details.

Contents:

Question: How do you access the User Settings page through the consolidation period?
Answer: To access the User Settings page through the consolidation period, click either the Current or Reference consolidation period on top of the page.

Question: What is indicated by a red lock on the User Settings page?
Answer: A red lock on the User Settings page indicates a locked consolidation period.

Question: How do you change the application language?
Answer: To change the application language, access the User Settings page through your Profile icon, go to the Application Language field, select another language from the drop-down, and click to confirm.

Question: How can you start a task from the Workflow page?
Answer: To start a task from the Workflow page, select the company from the table, select the task in the Task list field, and click to navigate to the task page.

Question: What happens when you reject a task on the Workflow page?
Answer: When you reject a task on the Workflow page, an automatic email is sent to the approver/administrator.

Question: How do you generate a status report?
Answer: To generate a status report, go to the Status Report panel, select the output file type, and click to generate the report.

Question: What is the purpose of the Status Board?
Answer: The purpose of the Status Board is to provide an at-a-glance view of all companies in the consolidation, including the status of the data entry process for each company.

Question: How do you change a company's data entry status?
Answer: To change a company's data entry status, select the company, and in the Change status panel, select the new status and click to change the status.

Question: What information does the Documents Repository store?
Answer: The Documents Repository stores collaborative documents accessible to all authorized users, where administrators can organize, maintain, and govern the stored documents.

Question: How do you filter consolidation periods using wildcards?
Answer: To filter consolidation periods using wildcards, use symbols like ? and * to narrow down the list of periods.

Question: What does locking a period do?
Answer: Locking a period restricts any modifications to the selected data type for that period across all companies within the group.

Question: What are the available options for data types when locking a period?
Answer: The available options for data types when locking a period are Closing Amounts, Intercompany, Flows, Intercompany Flows, Analytical Dimensions, and Adjustments.

Question: How do you unlock data in a consolidation period?
Answer: To unlock data in a consolidation period, go to the Define Period Locking page, click the data to unlock, clear the lock date, and click to confirm.

Question: How do you access the Exchange Rates page?
Answer: To access the Exchange Rates page, click Group > Exchange Rates.

Question: How do you enter currency exchange rates for a consolidation period?
Answer: To enter currency exchange rates for a consolidation period, select the consolidation period and currency, enter the Close, Average, and Month currency rates, and click to save changes.

Question: What is the purpose of the Exchange Rates page?
Answer: The purpose of the Exchange Rates page is to enter, import, or export currency exchange rates for a consolidation period.

Question: What are the file formats for importing exchange rates?
Answer: The file formats for importing exchange rates include CSV, Excel, ECB (SDMX European Central Bank), and NBS (SDMX National Bank of Slovakia).

Question: How do you import exchange rates from an Excel file?
Answer: To import exchange rates from an Excel file, specify the Sheet Name and Currency, upload the file, view it, and transfer the data into the database.

Question: How do you export currency exchange rates?
Answer: To export currency exchange rates, go to the Export Exchange Rates page, select the period, choose the file format (XLS or CSV), specify parameters, and click to export the rates.

Question: What information is displayed on the Companies page?
Answer: The Companies page displays a list of available companies on the left-side panel and details of any selected company, including code, description, and country code, on the right-side panel.

Question: How do you add a new company?
Answer: To add a new company, go to the Companies page, click the add icon, enter the required information, and save.

Question: What is the purpose of the Companies Additional Info page?
Answer: The purpose of the Companies Additional Info page is to enable users to add supplementary details for companies, which administrators can create and users can input corresponding values for their associated companies.

Question: How do you access the Companies Additional Info page?
Answer: To access the Companies Additional Info page, click Group > Companies and then click the add icon.

Question: How do you create the group structure?
Answer: To create the group structure, go to the Group Structure page, input and manage direct relationships among entities, determining the appropriate consolidation method.

Question: What information is shown in the Shareholders tab on the Group Structure page?
Answer: The Shareholders tab on the Group Structure page shows the number of shares held by other companies in the group.

Question: What information is shown in the Participations tab on the Group Structure page?
Answer: The Participations tab on the Group Structure page shows the number of shares held in other companies.

Question: How do you update the consolidation method and percentages?
Answer: To update the consolidation method and percentages, go to the Group Structure page, save changes, and proceed with caution as it can have significant consequences.
